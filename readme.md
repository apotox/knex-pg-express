## Knex Query Builder + Postgres Database + ExpressJS

___

### Using
- typescript
- express endpoints
- unit tests + mocks
- vitest setup inMemory pg database
- migrations
- github actions


### Install
- `yarn install`

### generate types from database schema
- `yarn generate:types`

### create new migrations
- `yarn migrate:make <migration name>`

### sync/run migration
- `yarn migrate:run`

### build
- `yarn build`

### tests
- `yarn test`
